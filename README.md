## 🚀 How to Use

1. Open the [live site](https://monarchbaweja1.github.io/placements_bda_27)
2. Use the **sidebar** to navigate between sections
3. In **Companies** — use filter buttons (Consulting / BFSI / Tech / FMCG) to narrow down
4. Click any company card to expand its questions (Technical / Behavioural / Case Study tabs)
5. Use **⬇ Download as PDF** buttons to save question sets offline
6. **Cheatsheets** tab has quick reference for Python, SQL, ML, Statistics, DL, Power BI
